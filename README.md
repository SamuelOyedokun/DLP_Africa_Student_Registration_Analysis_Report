# DLP Africa Student Registration Analysis

## Overview
Analysis of 40 student registrations across 8 courses 
for DLP Africa (May 2026 Cohort).

## Tools Used
Microsoft Excel — COUNTIF, VLOOKUP, HLOOKUP, 
INDEX/MATCH, Pivot Tables, Dashboard

## Key Findings
- 40 students across 8 courses
- Data Analysis most popular course (27.5%)
- Gender split: 60% Male, 40% Female
- Average age: 23.3 years
- Peak registration: 5 May (8 students)
